Aplicación del Clima Asistida por IA

Esta aplicación recibe la entrada del usuario, realiza una solicitud HTTP a Open-Meteo y muestra los datos meteorológicos en un formato amigable. Además, incluye manejo de errores, soporte para múltiples ciudades y pruebas rápidas.

Puedes consultar una ciudad o varias separadas por coma. Ejemplo: Tokio, Madrid, Bogotá.

Ingresa una ciudad para comenzar.

Resultados del clima

La app muestra temperatura, velocidad del viento, humedad y una descripción breve del clima.

Qué incluye esta aplicación

  • Llamada API funcional para obtener datos meteorológicos reales.
  • Manejo de errores para ciudad inválida, campo vacío o fallas de red.
  • Funcionalidad avanzada: consulta del clima para múltiples ciudades.
  • Interfaz limpia con tarjetas para mostrar la información.
  • Pruebas rápidas integradas para validar funciones clave.
